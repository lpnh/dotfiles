# About

For those seeking documentation, the [ArchWiki](https://wiki.archlinux.org) is
the best resource available. In fact, it's so good that it can be a perfect
trap for a curious mind. My goal here is to gather the information I find
relevant for my own use case, allowing me to setup and maintain a simple and
aesthetic [Arch Linux](https://archlinux.org/)
[Wayland](https://wayland.freedesktop.org/) environment while minimizing the
chances of falling down a rabbit hole.
